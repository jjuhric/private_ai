# Deployment Rules
- **Current Setup**: The Raspberry Pi auto-update deployment is configured using **Tailscale Funnel**.
- **Future Plan**: Transition to **Cloudflare Tunnels** when migrating to custom domains.

# Weather Formatting Rules
- **Feels Like Temperature**: In all weather reports and forecasts (including hourly and consolidated 3-hour forecasts), you MUST always include the "feels like" temperature (or expected apparent temperature) alongside the standard temperature.
